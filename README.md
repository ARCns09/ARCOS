# ARCOS

ARCOS is a modern Linux distribution based on Fedora KDE.

## Vision

ARCOS focuses on:

* Performance
* Simplicity
* Gaming
* Development
* Beautiful KDE Plasma experience

### Philosophy

* Keep Fedora's excellent default applications.
* Improve the operating system, not replace it.
* Stay close to upstream Fedora.
* Deliver a polished experience with minimal bloat.

## Roadmap

* Bootable ARCOS ISO
* ARCOS branding
* KDE customization
* Gaming enhancements
* Custom display scaling
* Future ARC applications
